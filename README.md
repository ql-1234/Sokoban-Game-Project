# 推箱子小游戏
## 项目概述

这是一个基于C语言开发的推箱子小游戏，实现了推箱子玩法并增加了体力系统、撤销/重做功能和自定义关卡等特色功能。

## 功能特点

- 6种地图模式：4种预设难度 + 自定义地图 + 继续上次游戏
    
-  体力系统：不同难度有不同的体力限制机制
    
-  操作历史：支持撤销(Z)/重做(Y)功能
    
-  游戏存档：自动保存游戏进度
    
-  自定义关卡：支持创建5×5到100×100的地图
    

## 编译与运行

```bash
// 编译命令
gcc -o sokoban sokoban.c

// 运行游戏
./sokoban
```
## 游戏操作说明

|按键|功能|
|---|---|
|W|向上移动|
|A|向左移动|
|S|向下移动|
|D|向右移动|
|Z|撤销上一步操作|
|Y|恢复被撤销的操作|
|Q|退出游戏（可存档）|
## 游戏符号说明

| 符号  | 含义       |
| --- | -------- |
| #   | 墙壁       |
| @   | 玩家       |
| $   | 箱子       |
| .   | 目标点      |
| !   | 玩家站在目标点上 |
## 文件说明

- `sokoban.c`：游戏主程序源代码
    
- `data/` ：地图数据文件夹
    - `难度：简单.txt`
        
    - `难度：一般.txt`
        
    - `难度：困难.txt`
        
    - `难度：地狱.txt`
        
    - `上次的游戏.txt`
        
其中，“难度：简单.txt”、“难度：一般.txt”、“难度：困难.txt”、“难度：地狱.txt”已经设置好，无需改动，“上次的游戏.txt”会根据中途退出时保存的游戏数据设置自身的参数

## 系统要求

- **操作系统**：Windows/Linux/macOS
    
- **编译器**：GCC 11或更高版本
    
- **文件编码**：GB 2312
